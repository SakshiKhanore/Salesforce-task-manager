# Salesforce Task Manager

A custom CRM-style task allocation and tracking application built on Salesforce Developer Edition.

## Features
- Custom Objects: `Task__c` and `Employee__c` with relational User Lookup fields
- Task fields: Name, Due Date, Status (To Do / In Progress / Done), Priority (High / Medium), Assigned To, Description
- Employee fields: Name, Email, Department, Contact
- 3 custom Salesforce Reports:
  - Tasks By Due Date
  - Tasks By Priority
  - Tasks By Status
- Task Manager Dashboard with 3 live panels (line chart, bar chart, donut chart)

## Tech Stack
- Salesforce Developer Edition
- Salesforce Lightning App Builder
- Salesforce Reports & Dashboards

## What I Learned
- Custom object and field creation in Salesforce
- Relationship modelling using Lookup fields
- Building and filtering Salesforce Reports
- Composing multi-component Dashboards for operational monitoring
- App creation and navigation setup in Lightning Experience
